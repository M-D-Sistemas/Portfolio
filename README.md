# M&D Sistemas Portfolio

Landing page tipo SPA para la agencia M&D Sistemas, construida con React, Vite y Tailwind CSS.

## Descripcion

Este proyecto presenta una web comercial orientada a clientes B2B, con un enfoque visual oscuro, limpio y responsivo. Incluye secciones de hero, tecnologia, proyectos, presentacion del equipo y contacto.

## Tecnologias

- React
- Vite
- Tailwind CSS

## Desarrollo local

1. Instalar dependencias:

```bash
npm install
```

2. Levantar el entorno de desarrollo:

```bash
npm run dev
```

3. Generar build de produccion:

```bash
npm run build
```

4. Previsualizar el build:

```bash
npm run preview
```

## Estructura base

- `src/App.jsx`: composicion principal de la landing.
- `src/components/`: componentes de seccion reutilizables.
- `src/data/projects.js`: datos mock de proyectos.
- `src/index.css`: estilos globales y configuracion de Tailwind.

## Secciones

- Hero con CTA principal.
- Tech Stack con grilla de tecnologias.
- Projects Gallery con tarjetas dinamicas.
- About Us con descripcion del equipo.
- Contact Form con estado en React.

## Propuesta

El objetivo es comunicar claridad tecnica, confianza y capacidad de entrega para convertir visitas en consultas comerciales.
